# AMH Eurostoxx50 Test

This repository contains the code, datasets and results used for the empirical analysis of my bachelor thesis.

The analysis tests the Adaptive Markets Hypothesis on the European stock market, using the EURO STOXX 50 as the target index and a set of macro-financial variables as inputs.

## Repository structure

- `notebooks/`: contains the notebooks used for the empirical analysis.
  - `01_Data_Preprocessing.ipynb`: imports, cleans and prepares the data.
  - `02_AMH_Europe_Test.ipynb`: performs the empirical analysis and the AMH testing procedure.

- `data/`: contains the datasets used in the analysis.
  - `raw/`: original datasets before preprocessing.
  - `processed/`: final datasets used as input for the models.

- `results/`: contains the output files generated by the empirical analysis, including prediction errors, statistical test results and summary tables.

## Main objective

The aim of the analysis is to study whether changes in the economic-financial environment can be identified through the prediction errors of machine learning models, and whether the market shows a gradual adaptation process after periods of high uncertainty.

## Models

The empirical analysis uses two machine learning models:

- Multilayer Perceptron, used as a benchmark model.
- Long Short-Term Memory network, used as the main model to capture temporal relationships in the data.

## Thesis context

This repository was created as supplementary material for a bachelor thesis on the Adaptive Markets Hypothesis and its empirical application to the European stock market.
